# CET-6 六级全能复习平台

一个纯静态的浏览器端六级备考工具，无需安装任何依赖，直接双击 `index.html` 即可使用。

收录 2023–2025 年共 18 套六级真题，涵盖听力、阅读、翻译、写作四大题型，并附带 4000 词汇闪卡系统与模拟测试功能。

---

## 功能概览

| 模块 | 内容 |
|------|------|
| 听力 | 18套真题音频 + 交互式答题 + 自动批改 |
| 阅读 | 选词填空 / 长篇阅读 / 仔细阅读，含答案解析 |
| 翻译 | 18套真题原文 + 参考译文对照 |
| 写作 | 18套真题题目 + 写作模板 + 开头句参考 |
| 词汇 | 4000词闪卡，支持标记生词、随机/顺序模式 |
| 模拟 | 随机抽题生成模拟测试，自动评分 |

---

## 快速开始

1. 克隆或下载本仓库
2. 直接用浏览器打开 `index.html`（无需本地服务器）
3. 通过顶部导航栏切换各功能模块

> 兼容 Chrome / Edge / Firefox 最新版本。Safari 也可正常使用。

---

## 各模块使用说明

### 听力

1. 点击顶部导航「听力」
2. 在「真题练习」卡片中选择年份和套次
3. 点击播放按钮收听音频
4. 切换 Section A / B / C 标签作答
5. 点击「检查答案」查看批改结果，答错的题目会高亮显示正确答案

**说明：** 每年第二套和第三套共用同一份音频，选项顺序略有不同。

---

### 阅读

1. 点击顶部导航「阅读」
2. 选择年份和套次
3. 通过三个标签切换题型：
   - **Section A 选词填空**：点击空白处选择词库中的单词，点击「检查答案」批改
   - **Section B 长篇阅读**：阅读长文后，为每个句子选择对应段落字母（A–N），点击「检查答案」批改
   - **Section C 仔细阅读**：阅读两篇短文并回答选择题，点击「检查答案」批改

---

### 翻译

1. 点击顶部导航「翻译」
2. 选择年份和套次
3. 在文本框中输入译文
4. 点击「显示参考译文」对照官方参考答案

---

### 写作

1. 点击顶部导航「写作」
2. 在「真题练习」区域选择年份和套次
3. 题目和开头句参考会自动显示
4. 在文本框中完成写作
5. 「写作技巧」区域提供常用句型和模板参考

---

### 词汇

1. 点击顶部导航「词汇」
2. 闪卡正面显示英文单词，点击翻转查看中文释义
3. 点击「标记生词」将单词加入生词本
4. 右上角可切换「随机模式」或「顺序模式」
5. 「生词本」标签页集中复习已标记的单词

---

### 模拟测试

1. 点击顶部导航「模拟」
2. 点击「开始测试」生成随机题目
3. 完成所有题目后点击「提交」查看得分

## 技术说明

- **纯静态**：无后端、无构建工具、无 npm，直接 `file://` 协议运行
- **数据层**：所有题目数据以全局 JS 常量形式存储在 `data/` 目录下
- **模板挂载**：页面 HTML 以 `<script type="text/html">` 内联，由 `app.js` 在初始化时注入对应容器，绕过 `file://` 协议的 fetch 限制
- **音频共享**：2/3套共用同一 mp3，通过 `audio` 字段映射

---

## 数据来源

- 真题题目：从历年六级真题 PDF 提取
- 词汇数据：新东方六级词汇 + 周计划词汇表合并去重
- 参考译文：官方参考答案

## 下一步计划

### 模拟测试质量提升（优先级最高）

当前模拟测试从固定题库随机抽题，题目质量和覆盖度有限。计划改进方向：

- **按题型分配比例**：严格按照真实六级考试结构出题（听力 35%、阅读 35%、翻译 15%、写作 15%）
- **跨套次随机抽题**：从 18 套真题中随机抽取，避免重复刷同一套题
- **难度标注与分层**：对题目标注难度等级，支持按难度筛选练习
- **错题本联动**：答错的题目自动加入错题本，支持专项复习
- **计时功能**：加入倒计时，模拟真实考试时间压力（全卷 130 分钟）
- **成绩趋势图**：记录历次模拟得分，可视化进步曲线
