# CET-6 六级全能复习平台

一个纯静态的浏览器端六级备考工具，无需安装任何依赖，直接双击 `index.html` 即可使用。

收录 2023–2025 年共 18 套六级真题，涵盖听力、阅读、翻译、写作四大题型，并附带 4000 词汇闪卡系统与模拟测试功能。

---

## 功能概览

| 模块 | 内容 |
|------|------|
| 听力 | 18套真题音频 + 交互式答题 + 自动批改 + 6组高频场景词汇 |
| 阅读 | 选词填空 / 长篇阅读（下拉选段落）/ 仔细阅读，含答案解析 |
| 翻译 | 18套真题原文 + 参考译文对照 + 5组高频主题词汇 |
| 写作 | 18套真题题目 + 写作思路 + 高分句型速查 |
| 词汇 | 2487词闪卡，支持生词本、已掌握管理与撤回 |
| 备考技巧 | 四大题型答题技巧与积累方法，基于2023–2025真题规律 |
| 模拟 | 随机抽题生成模拟测试，自动评分 |

---

## 快速开始

1. 克隆或下载本仓库
2. 直接用浏览器打开 `index.html`（无需本地服务器）
3. 通过顶部导航栏或首页快速导航切换各功能模块

> 兼容 Chrome / Edge / Firefox 最新版本。Safari 也可正常使用。

---

## 各模块使用说明

### 听力

1. 点击顶部导航「听力」
2. 在「真题练习」卡片中选择年份和套次
3. 点击播放按钮收听音频
4. 切换 Section A / B / C 标签作答
5. 点击「检查答案」查看批改结果，答错的题目会高亮显示正确答案

**说明：** 每年第二套和第三套共用同一份音频，选项顺序略有不同。

---

### 阅读

1. 点击顶部导航「阅读」
2. 选择年份和套次
3. 通过三个标签切换题型：
   - **Section A 选词填空**：点击空白处选择词库中的单词，点击「检查答案」批改
   - **Section B 长篇阅读**：阅读长文后，用下拉菜单为每条陈述选择对应段落字母，点击「检查答案」批改
   - **Section C 仔细阅读**：阅读两篇短文并回答选择题，点击「检查答案」批改

---

### 翻译

1. 点击顶部导航「翻译」
2. 选择年份和套次
3. 在文本框中输入译文
4. 点击「显示参考译文」对照官方参考答案

---

### 写作

1. 点击顶部导航「写作」
2. 在「真题练习」区域选择年份和套次
3. 题目和开头句参考会自动显示
4. 在文本框中完成写作（2023年起全部为续写题型）
5. 「写作思路」按钮提供结构建议与高分句型参考

---

### 词汇

1. 点击顶部导航「词汇」
2. 闪卡正面显示英文单词，点击翻转查看中文释义
3. 翻转后可点击「✓ 已掌握」将单词移出练习池，或「☆ 加入生词本」标记
4. 「已掌握」按钮可查看所有已掌握单词，支持逐个撤回
5. 「生词本」按钮集中复习已标记的单词

---

### 模拟测试

1. 点击顶部导航「模拟测试」
2. 选择专项或综合测试
3. 完成所有题目后点击「提交」查看得分

## 技术说明

- **纯静态**：无后端、无构建工具、无 npm，直接 `file://` 协议运行
- **数据层**：所有题目数据以全局 JS 常量形式存储在 `data/` 目录下
- **音频共享**：2/3套共用同一 mp3，通过 `audio` 字段映射

---

## 数据来源

- 真题题目：从历年六级真题 PDF 提取
- 词汇数据：新东方六级词汇 + 周计划词汇表合并去重
- 参考译文：官方参考答案

## Next Steps

### 完善模拟测试功能（优先级最高）

当前模拟测试从固定题库随机抽题，题目覆盖度和真实度有限。计划改进方向：

- **按真实比例出题**：严格按照六级考试结构分配题型（听力 35%、阅读 35%、翻译 15%、写作 15%）
- **跨套次随机抽题**：从 18 套真题中随机抽取，避免重复刷同一套题
- **计时功能**：加入倒计时，模拟真实考试时间压力（全卷 130 分钟）
- **错题本联动**：答错的题目自动加入错题本，支持专项复习
- **成绩趋势图**：记录历次模拟得分，可视化进步曲线

See [CHANGELOG.md](CHANGELOG.md) for a full history of changes.
